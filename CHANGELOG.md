# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-01-14

### Added
- Initial release
- `/cowork` command for autonomous file operations
- `cowork_execute` tool for complex tasks
- `cowork_analyze` tool for read-only code analysis
- Streaming output with real-time thinking process display
- Permission confirmation for all file operations
- Oh My OpenCode integration support
- macOS notification on task completion
- Detailed statistics (time, turns, tool calls, cost, tokens)
